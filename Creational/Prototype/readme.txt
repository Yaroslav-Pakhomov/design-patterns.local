Прототип
Prototype

Прототип - порождающий шаблон проектирования, который необходим для копирования объекта, не вдаваясь в детали его реализации. В нормальном мире реализуется через написание интерфейса с методом clone, который потом реализует класс прототипа, однако в PHP для этого есть ключевое слово clone, используемое в конструкции "$anotherObject = clone $object;". В результате - получаем "клонированный" объект. В php присутствует магический метод __clone(), который необходимо реализовать и с помощью его можно произвести какие-либо действия с клонированным объектом (например что-то сделать с его свойствами).

Обратите внимание на то, что метод __clone() вызывается уже после копирования объекта!

Так же важно то, что если среди свойств объекта есть другие объекты (в PHP в переменной содержащей объект находится ссылка на этот объект) - все "клоны" будут ссылаться на один и тот же объект.
