Снимок
Memento

Снимок - поведенческий шаблон проектирования, который позволяет сохранять состояния объекта, возвращать их к предыдущему или предпредыдущим состояниям (отменять изменения), при этом не раскрывая подробности реализации. Для реализации шаблона нам потребуется объект, состояние которого мы будем сохранять, объект снимка, содержащий необходимые для восстановления данные, а так же объект - управляющий всем процессом, через него и будет осуществляться вся работа.

1. Создадим класс StringObject, хранящий строку, в нем создадим метод createSnapShot, инициализирующий и возвращающий объект снимка

2. Для класса снимка заведем интерфейс ISnapShot.

3. Реализуем интерфейс ISnapShot в классе снимка StringSnapShot объекта StringObject.

4. Теперь создадим класс StringManager для управления всей этой структурой.

5. Протестируем получившийся код.
